Para descargar como ZIP el proyecto y ejecutarlo: 
1. Cambiar el nombre de carpeta descomprimida a "ProyectoFinal" y colocarla el htdocts
2. En la terminal de proyecto escribir el siguiente comando: composer install
3. Pegar el archivo .env en la raiz del proyecto junto al resto de carpetas
4. En la terminal de proyecto escribir el siguiente comando:  
php artisan key:generate
5. En la terminal de proyecto escribir el siguiente comando: 
composer require laravel/ui
6. En la terminal de proyecto escribir el siguiente comando: 
php artisan ui react
7. En la terminal de proyecto escribir el siguiente comando:  
npm install
8. En la terminal de proyecto escribir el siguiente comando: 
npm install emailjs-com
9.  En la terminal de proyecto escribir el siguiente comando:
npm install @paypal/react-paypal-js
10.  En la terminal de proyecto escribir el siguiente comando:
npm install @react-google-maps/api
12. En la terminal de proyecto escribir el siguiente comando:
npm run dev
